O que é SASS e como funciona?

O SASS (Syntactically Awesome Style Sheets) é uma extensão do CSS que oferece uma forma mais poderosa e elegante de escrever estilos para páginas web. Ele introduz recursos como variáveis, mixins (fragmentos de código reutilizáveis), placeholders e funções, permitindo uma escrita mais eficiente e modular de estilos.

O SASS é compilado para CSS, o que significa que, após escrevermos nossos estilos no SASS, precisamos compilá-los em arquivos CSS para que os navegadores possam interpretá-los corretamente.

Neste exemplo, estamos utilizando variáveis, mixins e placeholders do SASS para estilizar elementos HTML de forma mais flexível e organizada.
